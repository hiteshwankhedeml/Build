# Scenario - Script Task

**Requirement:**

* Design a process to automate the shopping cart for products available at the global API of our organization
* As part of the input user can request a discount % to each of the product
* The approval should be sent parallely to VP with a table showing products and total value with discount
* Once both the manager and VP approves the request, an approval email will be sent to the requestor
* If VP does not take action on shopping cart request for more than 5minutes, we should proceed without approval
