# SAP UI5

* Framework to develop responsive web applications which can run on any device, any platform
* Its SAPs latest UI technology to build Fiori applications
* We can develop an UI5 application with BAS

Main aspects:

* component.js: a starting file for launching and initializing our app
* View: The application user interface to display controls to the user
* Controller: Where we write all processing logic
* Model: The data related to the application
