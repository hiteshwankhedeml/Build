# Steps to configure Desktop Agent

1. Download the Desktop Agent - RPA agent - [https://tools.hana.ondemand.com/#cloud](https://tools.hana.ondemand.com/#cloud)
2. In control tower ⇒ Agent ⇒ Register ⇒ it will give URL
3. Install, when we launch it, it will ask for tenant ⇒ Add the above given URL
4. Activate the agent ⇒ Also make the mode as Unattended
5. Ask for BTP login
6. Once done, it will be shown in Agent in Control tower
7. In Environment, add the Agent now
8. Add chrome browser ⇒ SAP BPA extension&#x20;

In organization for productive agent, we need to follow the guide, we will be needing S id to download from marketplace
