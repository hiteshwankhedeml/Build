# The destination sap\_process\_automation\_service

* For showing and reading workitems in My inbox, it needs to communicate with the bpa
* BPA has business processes, when business processes is executed it creates instances
* This instances are in memory of business process
* The communication between Workzone and this workitems will be done using destination
* As per SAP for this destination sap\_process\_automation\_service needs to be created
* This destination will have information about the build process automation service ⇒ For this it will use service key of the bpa instance
* Subaccount ⇒ Connectivity ⇒ Destinations
* Enter destination name, enter service key details and properties
* Once destination is created, we can check connection also
* [https://help.sap.com/docs/build-process-automation/sap-build-process-automation/configure-sap-build-process-automation-destinations](https://help.sap.com/docs/build-process-automation/sap-build-process-automation/configure-sap-build-process-automation-destinations)
