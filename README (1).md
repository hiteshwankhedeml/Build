# SAP Build

* Earlier it was used as prototyping tool
* User friendly low code no code platform within BTP
* Designed to empower users to create, customize and enhance applications, automate workflow and build business sites without the need for advanced coding skills
* It includes 3 major components
  * SAP Build Apps: Create Apps
  * SAP BPA: To Automate processes using workflows and robotic process automation
  * SAP Build workzone: To create business sites, formerly known as fiori launchpad
