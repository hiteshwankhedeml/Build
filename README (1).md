# 🟢 SAP Build

* Earlier it was used as prototyping tool
* <mark style="color:purple;background-color:purple;">**User friendly low code no code platform within BTP**</mark>
* <mark style="color:purple;background-color:purple;">**Designed to empower users to create, customize and enhance applications, automate workflow and build business sites without the need for advanced coding skills**</mark>

  * <mark style="color:purple;background-color:purple;">**SAP Build Apps:**</mark> <mark style="color:purple;background-color:purple;"></mark><mark style="color:purple;background-color:purple;">Create Apps</mark>
  * <mark style="color:purple;background-color:purple;">**SAP BPA:**</mark> <mark style="color:purple;background-color:purple;"></mark><mark style="color:purple;background-color:purple;">To Automate processes using workflows and robotic process automation</mark>
  * <mark style="color:purple;background-color:purple;">**SAP Build workzone:**</mark> <mark style="color:purple;background-color:purple;"></mark><mark style="color:purple;background-color:purple;">To create business sites, formerly known as fiori launchpad</mark>
