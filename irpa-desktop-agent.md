# iRPA Desktop Agent

* Component of BPA that is installed locally on user desktops
* It executes automation projects that launch and run applications of various kinds, reads information from screen, enters data, clicks options and processes data
* Projects are assigned to tenants running on a desktop agent
* Using system tray we can see what it is doing all the time
* When it is installed, the desktop agent is configured to start at windows login automatically
* You should not change the setting because your agent might be assigned background jobs at nay time
* The first time we launch the desktop agent, we will be asked to login into btp
