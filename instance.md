# Instance

* We create a process as a citizen developer
* This is deployed on BTP ⇒ Each process will have datacenter.subaccount.project.process
* There will be some build or fiori application which will trigger the process
* To trigger this we need BPA service instance
* This is known as API trigger business process
* Sub account ⇒ Instance and subscription ⇒ Create
* **SAP Build Process Automation ⇒ Select Instance this time**
* Once instance is created, then we can create service key
* This key can be used to trigger BPA
