# Steps

**Action Step:**

1. Create a destination for the API(Service) we want to call
2. Create an action for the destination
3. Control tower ⇒ Destination ⇒ Map it with our Action Project
4. Observe the outputs and data types if match with results
5. Test the action
6. Once tested, save, release, publish

**Process:**

1. Create process
2. Define input parameter and API trigger
3. Add action and system created a list data type automatically
4. Use the data type to create variable and discount and total amount
5. Map the input and output with action, input process and custom variable
6. Add script task to calculate total and discount amount
7. Add parallel branch to execute approval step to manager and VP
8. Create deadline monitoring for VP approval flow
